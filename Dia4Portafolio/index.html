Mis Proyectos

Travelmania

Esta página fue creada utilizando Flexbox para aprender los conceptos básicos de maquetación en CSS. Ofrece una experiencia visual atractiva al presentar una variedad de destinos de viaje en un diseño flexible y responsivo. Los usuarios pueden explorar diferentes destinos y obtener información detallada sobre cada experiencia de viaje de manera intuitiva.

Tecnologías Utilizadas:HTML, CSS, Git y GitHub.

Wanderlust

La agencia de viajes en línea se diseñó como una plataforma integral para ayudar a los usuarios a planificar sus viajes de manera eficiente. Ofrece una interfaz amigable que permite a los usuarios buscar destinos, comparar precios de vuelos y alojamiento, reservar actividades y crear itinerarios personalizados.

Tecnologías Utilizadas: HTML, CSS, JavaScript, Git y GitHub.
